# Nand2Tetris: Building a Modern Computer from First Principles

This repository contains my solutions and implementations for all 12 projects of the **Nand2Tetris** course, where a complete computer system is built from the ground up, starting with a single NAND gate and progressing to a functioning software stack.

## Overview

The Nand2Tetris course explores how modern computers work by constructing every layer of a computing system from first principles. Through a series of projects, I designed and implemented both hardware and software components of the Hack computer platform.

Key topics covered include:

* Digital Logic Design
* Computer Architecture
* Assembly Language Programming
* Virtual Machines
* Compiler Construction
* Operating Systems
* Systems Programming

## Projects Completed

### Hardware Layer

* Boolean Logic Gates
* Arithmetic Logic Unit (ALU)
* Registers and Memory Units
* Program Counter (PC)
* CPU Design
* Complete Hack Computer Architecture

### Software Layer

* Hack Assembler
* Virtual Machine (VM) Translator
* Jack Programming Language Compiler
* Operating System Libraries

## Learning Outcomes

Through this project, I gained practical experience in:

* Designing digital circuits using Hardware Description Language (HDL)
* Understanding CPU execution and memory management
* Translating high-level programs into machine-executable instructions
* Building language-processing tools such as assemblers and compilers
* Developing operating system abstractions from scratch
